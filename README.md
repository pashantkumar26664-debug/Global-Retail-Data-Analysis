# Global Retail Revenue Analysis Dashboard

## 📌 Project Overview
This project involves analyzing a dataset of over 500,000 transactional records for a UK-based online retail company. The goal was to uncover actionable insights for the CEO and CMO regarding seasonal revenue trends, international expansion opportunities, and VIP customer retention.

## 🛠️ Tools Used
* **Data Visualization & Analysis:** Tableau
* **Data Cleaning:** Tableau (Handling null values, excluding anomalies, creating calculated fields)
